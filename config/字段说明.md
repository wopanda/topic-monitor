# 配置字段说明

## 主文件
- 默认文件名：`config/topic-monitor-config.json`
- 首次使用：从 `config/topic-monitor-config.example.json` 复制一份出来再改

## 搜索策略

对读者来说，默认只需要记住一件事：
- **直接用默认自动模式**

系统会自动判断：
- 有 `TAVILY_API_KEY` → 效果增强
- 没有 `TAVILY_API_KEY` → 自动退回保底搜索

`search.provider / route / mode` 这些字段保留给开发调试使用，不建议作为读者主配置项。

## 读者最需要先改的字段

### topics[0].name
日报里展示的主题名。

### topics[0].keywords
真正驱动搜索的关键词数组。

### topics[0].preferredDomains
希望优先看的站点域名，例如 `github.com`、`larksuite.com`。

### topics[0].blockedDomains
明确要压掉的域名。

### output.finalItems
精选条数。

### output.watchItems
观察池条数。

### filters.excludeKeywords
排除词，例如广告、推广。

## 已做实测

### A. 搜索路由器：有 Tavily key
已实测：
```bash
python3 scripts/search_router.py "OpenClaw 用例" --provider auto --count 5 --pretty
```
结果：
- provider = `hybrid`
- resolved_route = `china + tavily`
- used_provider = `hybrid`

### B. 搜索路由器：无 Tavily key
已实测：
```bash
unset TAVILY_API_KEY
python3 scripts/search_router.py "OpenClaw 用例" --provider auto --count 5 --pretty
```
结果：
- provider = `china`
- resolved_route = `china-only`
- used_provider = `china`

这说明：
- **auto 行为已经符合目标**
- 没 key 时不会再因为 Tavily 缺失直接退出
